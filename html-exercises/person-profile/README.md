# Person Profile Page

## Exercise Overview

Build a basic HTML page with headings, paragraphs, an image, and a list of skills.

## Objectives

- Create a structured HTML document with proper semantic elements
- Use heading tags (h1, h2, h3, etc.) to organize content hierarchically
- Include paragraphs with biographical information
- Embed an image with appropriate alt text
- Create a list of skills using HTML list elements

## Key HTML Elements to Use

- `<h1>`, `<h2>`, `<h3>` - Headings
- `<p>` - Paragraphs
- `<img>` - Images
- `<ul>` or `<ol>` - Lists
- `<li>` - List items

## Suggested Structure

1. Main heading with the person's name
2. A paragraph introducing the person
3. An image (profile photo or placeholder)
4. A section with a heading for "Skills"
5. An unordered list of skills

## Tips

- Remember to include proper DOCTYPE and HTML structure
- Use semantic HTML5 elements where appropriate
- Add alt text to images for accessibility
- Consider adding CSS styling to make your profile page visually appealing
