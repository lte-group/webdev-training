# Simple Navigation Menu

## Exercise Overview

Create an unordered list with links to different sections of a page using anchor tags.

## Objectives

- Build a navigation menu using HTML list elements
- Create internal page links using anchor tags with fragment identifiers
- Structure a page with multiple sections
- Implement smooth scrolling navigation between sections

## Key HTML Elements to Use

- `<nav>` - Navigation container
- `<ul>` - Unordered list
- `<li>` - List items
- `<a>` - Anchor tags with href attributes
- `<section>` or `<div>` - Page sections with id attributes

## Suggested Structure

1. Create a `<nav>` element with an unordered list
2. Add list items with anchor links (e.g., `<a href="#about">About</a>`)
3. Create corresponding sections on the page with matching id attributes (e.g., `<section id="about">`)
4. Add content to each section

## Tips

- Use meaningful id names for sections
- Fragment identifiers in links should start with #
- Consider adding CSS to style the navigation menu (horizontal layout, hover effects)
- Test that clicking each link scrolls to the correct section
