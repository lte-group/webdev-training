# Embed Media

## Exercise Overview

Create a page that embeds a video, audio file, and iframe.

## Objectives

- Embed video content using HTML5 video elements
- Include audio content with proper controls
- Embed external content using iframes
- Understand media element attributes and controls

## Key HTML Elements to Use

- `<video>` - Video player
- `<audio>` - Audio player
- `<iframe>` - Embedded content
- `<source>` - Media source files

## Suggested Structure

1. Add a heading for the page
2. Embed a video with controls
3. Embed an audio file with controls
4. Embed external content using an iframe (e.g., YouTube video, map)

## Tips

- Use the `controls` attribute to show playback controls
- Include the `src` attribute or nested `<source>` elements for media files
- For video, consider adding `width` and `height` attributes
- Provide multiple source formats for better browser compatibility
- Add fallback text for browsers that don't support the media element
- For iframes, use `width` and `height` attributes to control size
- Consider accessibility by providing captions or transcripts
