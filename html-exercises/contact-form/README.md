# Contact Form

## Exercise Overview

Construct a form with input fields for name, email, message, and a submit button.

## Objectives

- Create a functional HTML form
- Use various input types for different data collection
- Implement proper form labels for accessibility
- Add a textarea for longer text input
- Include a submit button

## Key HTML Elements to Use

- `<form>` - Form container
- `<label>` - Input labels
- `<input>` - Input fields (type="text", type="email")
- `<textarea>` - Multi-line text input
- `<button>` - Submit button

## Suggested Structure

1. Create a `<form>` element
2. Add a name field with appropriate label
3. Add an email field with type="email"
4. Add a message textarea
5. Include a submit button

## Tips

- Connect labels to inputs using the `for` attribute and matching `id`
- Use the `required` attribute for mandatory fields
- Set appropriate `type` attributes for inputs (text, email)
- Consider adding placeholder text to guide users
- Use CSS to style the form for better user experience
- Add the `name` attribute to form elements for data submission
